# chaincode-price

This is the price asset chaincode for channel2 in the Hyperledger Fabric sample network. It manages price assets (id, productId, price, payer, payee) for Org1 and Org2.

- Main contract: AssetPriceContract (lib/assetPrice.js)
- Entry point: index.js

## Usage

Deploy this chaincode to channel2 using the Fabric test network scripts.
