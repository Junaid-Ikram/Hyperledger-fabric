# chaincode-product

This is the product asset chaincode for channel1 in the Hyperledger Fabric sample network. It manages product assets (id, name, quantity, owner) for Org1, Org2, and Org3.

- Main contract: AssetProductContract (lib/assetProduct.js)
- Entry point: index.js

## Usage

Deploy this chaincode to channel1 using the Fabric test network scripts.
